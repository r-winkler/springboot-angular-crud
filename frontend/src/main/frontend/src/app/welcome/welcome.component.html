Welcome

This is a simple Spring Boot Angular app demonstrating basic concepts like CRUD operations, Keycloak security, monitoring etc.

In order to make this application work properly, you need to have a Keycloak server configured and running. Furthermore, clone the spring-boot-admin-server project (https://github.com/r-winkler/spring-boot-admin-server.git) and start it as well.

Start frontend (standalone): 'npm start' in frontend project. http://localhost:4200

Start frontend (Spring Boot): http://localhost:8080
